QinglongScripts

青龙面板自动任务订阅仓库，包含多个常用服务的自动签到/任务脚本

GitHub 查看脚本

快速开始

1️⃣

添加订阅

在青龙面板中添加仓库订阅链接

https://github.com/CN-Grace/QinglongScripts.git
2️⃣

配置环境变量

在青龙面板的「环境变量」中配置各脚本所需的变量

查看 .env.example →
3️⃣

自动运行

青龙面板会根据脚本中的 cron 注释自动创建定时任务

订阅后自动执行
