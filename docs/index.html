🚀 快速开始

三步完成配置

简单几步，即可享受全自动签到体验

1

添加订阅

在青龙面板 → 订阅管理中添加仓库链接：
https://github.com/CN-Grace/QinglongScripts.git

2

配置环境变量

参考 .env.example 文件，在面板中添加各脚本所需的环境变量。记得将 notify.py 加入订阅黑名单。

3

坐等签到 ✅

面板会自动读取 # cron 注释创建定时任务，每天按时执行签到并推送通知，省心省力！
