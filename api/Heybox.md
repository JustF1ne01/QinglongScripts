# 小黑盒签到 API 文档

## 脚本信息

- **文件名**: `Heybox.py`
- **定时任务**: `0 9 * * *` (每天9:00执行)
- **功能**: 小黑盒每日签到

## 环境变量

| 变量名 | 必填 | 说明 |
|--------|------|------|
| `HEYBOX_COOKIE` | ✅ | 小黑盒 Cookie |

## API 接口

### 1. 签到

**请求:**
```
POST https://api.xiaoheihe.cn/bbs/account/checkin
```

**Headers:**
```
Cookie: {HEYBOX_COOKIE}
```

**响应:**
```json
{
  "status": "ok",
  "msg": "签到成功"
}
```

## 输出报告格式

```
🎯 小黑盒签到报告

✅ 签到成功

──────────────────
🕒 执行时间: 2026-05-29 09:00:00
```
