# SSL 证书监控 API 文档

## 脚本信息

- **文件名**: `SSL.py`
- **定时任务**: `0 8 * * *` (每天8:00执行)
- **功能**: SSL 证书到期监控

## 环境变量

| 变量名 | 必填 | 说明 |
|--------|------|------|
| `SSL_DOMAINS` | ✅ | 域名列表，逗号分隔 |
| `SSL_DAYS` | ❌ | 提醒天数，默认30天 |

## API 接本功能

### 获取 SSL 证书信息

**请求:**
```
GET https://{domain}:443
```

**使用 Python ssl 模块获取证书信息**

**响应:**
```python
{
  "notAfter": "2026-06-29 00:00:00",  # 证书到期时间
  "subject": "域名信息"
}
```

## 输出报告格式

```
🔒 SSL 证书监控报告

✅ example.com - 有效期至 2026-06-29 (剩余 30 天)
⚠️ test.com - 有效期至 2026-06-01 (剩余 2 天，即将到期)

──────────────────
🕒 执行时间: 2026-05-29 08:00:00
```
