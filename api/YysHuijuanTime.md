# 阴阳师绘卷活动时间获取 API 文档

## 脚本信息

- **文件名**: `YysHuijuanTime.py`
- **定时任务**: `0 0 * * *` (每天0:00执行)
- **功能**: 从阴阳师官网获取最新绘卷活动时间信息

## 功能说明

该脚本会：
1. 访问阴阳师官网公告列表页
2. 解析最新的维护更新公告
3. 提取绘卷活动的时间信息
4. 保存到 `yys_huijuan_time.json` 文件

## 数据结构

### yys_huijuan_time.json

```json
{
```

## 网页解析规则

### 公告列表页

**URL:** `https://yys.163.com/news/update/`

**解析正则:**
```python
r'<a[^>]+href="(?:(?:https?:)?//yys\.163\.com)?/news/update/(\d+/\d+_\d+\.html)"[^>]*title="([^"]*)"'
```

### 公告详情页

**内容提取选择器:**
```python
r'<div[^>]*class="artText"[^>]*>(.*?)</div>'
r'<div[^>]*id="NIE-art"[^>]*>(.*?)</div>'
r'<div[^>]*class="art-con"[^>]*>(.*?)</div>'
```

### 绘卷活动时间

**解析正则:**
```python
r'((?:SP|SSR|SR)?[一-龥]+追忆绘卷活动)'
r'(\d{1,2})月(\d{1,2})日维护后\s*[-—~]\s*(\d{1,2})月(\d{1,2})日(\d{1,2}:\d{2})'
```

**时间格式:** `X月X日维护后-X月X日23:59`
